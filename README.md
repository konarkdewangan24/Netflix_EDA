# Netflix_EDA
# Netflix Movies — Exploratory Data Analysis

This project is an exploratory data analysis of a Netflix movie dataset. The goal was to clean the raw data and study patterns in genres, audience ratings, movie popularity, and release trends. The analysis was carried out using Python along with pandas, Matplotlib, and Seaborn for data handling and visualization.

The dataset was first cleaned by removing missing values and checking for duplicates. The release date was converted into a proper year format, and unnecessary columns were removed to keep the data focused. Since many movies belong to more than one genre, the data was restructured so that each genre could be analyzed individually. Audience ratings were also grouped into simple categories to make comparisons easier.

The analysis answers five key questions: the most frequent genre on Netflix, how votes are distributed across genres, the movie with the highest popularity, the movie with the lowest popularity, and the year with the most releases. The findings are summarized with charts and a final report.

## Key Points

- Performed data cleaning: handled missing values, fixed data types, and removed duplicates.
- Restructured the dataset to analyze movies with multiple genres correctly.
- Categorized audience ratings into clear, comparable groups.
- Identified the most and least popular movies in the dataset along with their genres.
- Analyzed genre frequency and release trends over the years.
- Visualized all findings using bar charts and histograms.
- Summarized the complete analysis in a final report.
